# Plan Mode

## Instructions

- Make the plan extremely concise. Sacrifice grammar for the sake of concision.
- At the end of each plan, give me a list of unresolved questions to answer, if any.

## Why This Matters

1.  **Concision**: Prevents the agent from hallucinating complexity or getting lost in verbose explanations.
2.  **Unresolved Questions**: Forces the agent to identify "Known Unknowns" before writing code, preventing "Slop".
